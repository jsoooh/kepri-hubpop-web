  1. Java APP 서비스는 Cloud Foundry 플랫폼 상에서 Container를 생성하고, Container 내부에 Java 애플리케이션 동작에 필요한 Runtime 환경을 자동으로 구성함으로써, 애플리케이션 비즈니스 로직 구현 외에 개발자가 해야하는 기타 인프라 작업들에 대한 부담을 줄여줍니다.
  2. Java APP 서비스가 제공하는 Runtime 환경 서비스의 범위는 다음과 같으며, 사용자의 선택에 따라 자동으로 구성이 가능합니다.
    - Server Container(Tomcat, JBoss 등)
    - Runtime Framework(Spring, 전자정부 프레임워크, KDN Power Framework 등)
    - Language Runtime Environment(Open JDK)
  3. Java APP 서비스는 애플리케이션의 소스코드관리시스템(Git 등)과 연동하여 실행환경 Container 생성과 동시에 사용자가 개발한 애플리케이션을 자동으로 Container 상에 배치 할 수 있습니다.
  4. Java APP 서비스의 Scaling 기능을 통해 사용자는 개발한 애플리케이션의 분산처리 환경을 쉽게 구성 할 수 있으며, Disk와 Memory 설정 만으로 클라우드 기반의 서버 자원을 쉽게 할당 받아 활용할 수 있습니다.
  5. 애플리케이션 데이터베이스 생성 및 연결, 빌드/배포/테스트 자동화, 디버깅 등의 백-엔드 및 운영 관리 서비스와 IoT 등의 산업 특화 기능을 제공하는 서비스들과의 손쉬운 연결을 통해 사용자 애플리케이션이 제공해야할 비즈니스 요건을 빠르게 구축하고 Delivery 할 수 있습니다.
