GIS 활용하기

저장소 등록

공간 데이터 관리에 필요한 공간 DB(PostGIS)에 저장소를 등록해야 하며, 프로젝트별로 저장소를 등록하여 사용 할 수 있습니다

  • [GIS 데이터 관리]>[저장소 관리] 메뉴를 클릭하여 페이지를 이동합니다.
  • 저장소 추가 버튼을 클릭하여 페이지를 이동합니다.
  • 저장소 명을 입력합니다.
  • 저장 버튼을 클릭합니다.

레이어 등록

ShapeFile 업로드

저장소에 레이어 등록을 위해서는 ShapeFile 업로드 기능을 이용하여 등록합니다.

ShapeFile 업로드 시 각각의 레이어를 구성할 공간 데이터의 확장자 *.shp, *.dbf, *.shx 파일을 함께 업로드해야하며, 압축한 *.zip 형식의 파일로도 업로드할 수 있습니다.

  • [ShapeFile]>[ShapeFile 업로드] 메뉴를 클릭하여 페이지를 이동합니다.
  • 업로드하려는 ShapeFile을 Drag&Drop영역으로 Drag&Drop하여 업로드합니다.
  • ShapeFile 대기열 등록 알림창이 뜨면 확인 버튼을 클릭합니다.
  • ShapeFile 업로드 후에 해당 프로젝트의 공간데이터 적재가 완료되기 전까지는 Shape Loader 상태 페이지로 나타나게 되며 초기화 버튼을 클릭 시 공간데이터 적재가 취소됩니다.
레이어 추가

ShapeFile로 업로드한 레이어를 사용하기 위해서는 발행 단계를 거처야 사용할 수 있습니다.

  • [ShapeFile]>[ShapeFile 업로드] 메뉴를 클릭하여 페이지를 이동합니다.
  • 레이어 추가 버튼을 클릭하여 페이지를 이동합니다.
  • 해당 프로젝트에 ShapeFile로 업로드된 레이어 목록을 보여주며 해당 목록에서 미발행된 레이어를 확인할수 있습니다.
  • 미발행된 레이어 중 추가하고자하는 레이어 목록에서 레이어 추가 버튼을 클릭하여 페이지를 이동합니다.
  • 레이어 스타일을 선택하고 저장 버튼을 클릭합니다.

레이어 그룹 관리

레이어 그룹은 여러 레이어를 하나의 레이어처럼 그룹 지어 관리해주는 기능을 제공합니다.

  • [GIS 데이터 관리]>[레이어그룹 관리] 메뉴를 클릭하여 페이지를 이동합니다.
  • 레이어그룹 추가 버튼을 클릭하여 페이지를 이동합니다.
  • 레이어 그룹 명을 입력합니다.
  • 레이어 선택 버튼을 클릭합니다.
  • 레이어 그룹으로 설정하고자하는 레이어를 체크하고 확인버튼을 클릭합니다.
  • 레이어 목록는 체크한 레어이들이 출력되며, 화면표시 우선순위를 ↑, ↓ 버튼을 이용하여 간의 순서를 정의합니다.
  • 레이어 그룹 저장 버튼을 클릭합니다.

스타일 관리

레이어의 스타일을 추가할 수 있다

스타일을 추가하기 위해서 *.sdl 파일을 업로드하거나, xml형식의 사용자 정의 추가 기능을 지원합니다.

  • [GIS 데이터 관리]>[스타일 관리] 메뉴를 클릭하여 페이지를 이동합니다.
  • 스타일 추가 버튼을 클릭하여 페이지를 이동합니다.
  • 스타일 명을 입력하고, 스타일 작성 형식을 선택합니다.
  • 스타일 작성 형식이 xml작성 인 경우, 스타일 템플릿으로 제공되는 Poin, Line, Plygon 중 하나를 선택하여 xml형식의 스타일을 작성합니다.
  • 스타일 작성 형식이 SLD파일첨부 인 경우, 파일 선택 버튼을 클릭하고 해당 파일을 선택합니다.
  • 저장 버튼을 클릭합니다.

API 문서

클라이언트 개발 지원을 위해 만든 API 라이브러리 함수들을 설명하는 참조 문서입니다.

해당 API 함수를 클릭시 상세 설명이 Dialog Popup으로 나타나며 함수 설명 및 표현 예시 확인이 가능합니다.

API 활용예제

API 라이브러리를 이용하여 개발하기 전에 API 함수 사용에 대해 개발 테스트를 할 수 있도록 기능을 제공합니다.

Javascript를 작하여 실행 버튼을 클릭하면 Javascrip, HTML, CSS를 이용하여 결과화면을 보여줍니다.

WebMapClient

API 라이브러리의 함수를 이용하여 만든 예시 지도 입니다. 레이어 범례, 거리/면적 측정, 지도 확대, 계폐기 히트맵, 1점/2점 계통 추적, 전산화 번호 검색을 통한 해당 속성 위치 조회, 속성 조회, 확대 비율 등을 나타냅니다.
