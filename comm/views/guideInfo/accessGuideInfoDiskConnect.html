디스크(disk) 마운트(mount) 방법 안내

- 디스크 마운트란? 가상 서버에 추가(연결)된 디스크를 사용하기 위해 논리적인 디렉토리를 지정(연결)하는 것을 말합니다.

  • Linux 서버 디스크 마운트(mount)
  • Windows 서버 디스크 마운트(mount)

1. Linux 서버 디스크 마운트(mount)

Linux 계열(Ubuntu, CentOS) 서버에 연결된 디스크(Disk)를 사용하기 위한 마운트(mount) 방법을 안내합니다.

① ‘서버 SSH 접속’ 을 합니다.

② [lsblk] 명령을 입력하여 마운트정보를 확인합니다.

③ [sudo mkfs -t ext4 /dev/vdb] 명령을 입력하여 해당 disk(vdb)를 포맷합니다.

④ [sudo mkdir /mnt/data] 명령을 입력하여 마운트포인트(/mnt/data)를 생성합니다.

⑤ [sudo mount /dev/vdb /mnt/data] 명령을 입력하여 마운트포인트(/mnt/data)에 disk(vdb)를 마운트합니다.

⑥ [lsblk] 명령을 입력하여 마운트정보를 확인합니다

2. Windows 서버 디스크 마운트(mount)

Windows 서버에 연결된 디스크(Disk)를 사용하기 위한 마운트(mount) 방법을 안내합니다.

Windows 서버에 할당 가능한 디스크 사이즈는 총 2TB입니다. 초과시 오류가 발생할수 있으니 주의하시기 바랍니다.

1) 윈도우 서버 원격접속 → 작업 표시줄에서 ‘서버 관리자’ 실행

2) ‘서버 관리자’ → ‘도구(T)’ → ‘컴퓨터 관리’ 메뉴 선택

3) ‘컴퓨터 관리’창에서 서버와 연결된 디스크(할당되지 않음) 확인

4) ‘오프라인’ 상태의 디스크를 ‘온라인’으로 전환

① ‘디스크 1’ 영역에서 마우스 오른쪽 버튼 클릭

② 메뉴에서 '온라인(O)' 선택

5) 디스크 초기화

① ‘디스크 1’ 영역에서 마우스 오른쪽 버튼 클릭

② 메뉴에서 '디스크 초기화(I)' 선택

③ '디스크 초기화' 팝업에서 디스크 파티션 형식 선택 후 '확인' 버튼 클릭

-파티션 형식 : MBR(마스터 부트 레코드)(M)

6) 디스크 볼륨 구성 - 01

① ‘디스크 할당’ 영역에서 마우스 오른쪽 버튼 클릭

② 팝업 메뉴에서 '새 단순 볼륨(I)' 선택

7) 디스크 볼륨 구성 - 2

① ‘단순 볼륨 만들기’ 마법사 시작

② ‘단순볼륨크기(MB)(S)’ 설정 후 ‘다음’ 클릭

③ ‘드라이브 문자 할당’ 설정 후 ‘다음’ 클릭

④ 파티션 포맷에서 ‘볼륨 레이블’ 설정 후 ‘다음’ 클릭

⑤ ‘단순 볼륨 만들기’ 마법사 완료

8) 정상적으로 마운트(mount)된 디스크 확인
