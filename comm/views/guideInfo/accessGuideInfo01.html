Linux(CentOS, Ubuntu)서버 접속방법 안내

1. 사전 준비사항

가상서버에 접속하기 위해 필요한 클라이언트 도구(client tools)를 다운로드 합니다.

도구 설명 다운로드
Bitvise SSH Client(v8.19) 가상 서버에 접속하기위한 윈도우PC Client Tool 입니다.
Linux 서버에 접속할 수 있는 기능을 통합적으로 제공합니다.
(지원 프로토콜 : ssh, rdp, sftp, etc,..) 		Bitvise SSH Client.zip
다운로드 받은 파일은 사용자PC에서 원하는 위치에 압축해제 하시기 바랍니다.

가상서버(CentOS, Ubuntu 서버인 경우만 해당)에 접속하기 위한 인증 키 파일을 다운로드 합니다.

‘접속 인증키(KEY)’ 항목의다운로드 아이콘다운로드 아이콘을 클릭하여 사용자PC에 인증키 파일(.pem)을 저장합니다.

접속 인증키 파일은 가상서버에 접속하는 사용자를 인증하기 위한 유일한 수단입니다. 분실 시 생성한 서버에 접속할 수 없으므로 안전하게 보관하시기바랍니다.

2. 접속방법

접속도구를 통해 생성된 가상서버에 접속하는 방법을 안내합니다.

1) 접속도구 실행

사용자PC에 다운로드 받은 bitvise 디렉토리에서BvSsh 실행파일 아이콘파일을 실행합니다.

2) 접속정보 설정

① 생성한 서버의 접속정보를 참조하여 ‘접속 IP’와 ‘접속 계정’ 정보를 입력합니다.

서버 접속 정보

- 접속 IP :
192.168.200.67
- 접속 계정 :
ubuntu

② ‘서버 접속정보 설정 화면’에서 ‘Client key manager’ 버튼을 클릭합니다.

③ ‘인증키 관리 화면’에서 ‘Import’ 버튼을 클릭하여 미리 다운로드 받은 사용자 인증 키 파일(.pem)을 업로드 합니다.

④ ‘서버 접속정보 설정 화면’ 하단의 ‘Log in’ 버튼을 클릭합니다.

3) 서버 접속

‘호스트키 수락 화면’에서 ‘Accept and Save’ 버튼을 클릭하여 서버에 접속합니다.

3. 기타

1) 서버 접속정보(Profile) 저장

‘서버 접속정보 설정 화면’ 좌측 ‘Save profile as’ 버튼을 클릭하여 현재 설정된 서버 접속정보를 사용자PC에 저장합니다.

*저장된 접속정보는 ‘Load profile’ 버튼을 통해 언제든지 다시 불러올 수 있습니다.

2) 접속에 실패한 경우 확인 사항

① ‘서버 접속정보 설정 화면’에서 ‘Proxy settings’ 버튼을 클릭합니다.

② ‘프록시(Proxy)서버 설정 화면’에서 아래 정보가 정확하게 설정되어 있는지 확인합니다.
