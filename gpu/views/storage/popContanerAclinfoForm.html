컨테이너명 crossent-default-org
특정 사용자
모든 사용자
설명

KPAASTA 사용자:KPAASTA의 팀별 오브젝트 스토리지 접근정보를 (Access Key,Secret Access Key) 소유한 사용자

모든 사용자:인터넷에 접속할 수 있는 모든 사용자

FULL:컨테이너의 모든(read,write) 권한

READ:컨테이너 정보(생성일,버전,오브젝트 목록)조회만 가능

WRITE:컨테이너에 대해 오브젝트 등록,컨테이너 삭제,컨테이너 수정(OVER WRITE)가 가능함

READ-ACP:컨테이너에 대해 접근 권한 조회가 가능함

WRITE-ACP:컨테이너에 대해 접근 권한 변경이 가능함
