Object Storage

자유롭게 데이터를 저장 및 확인이 가능한 스토리지 입니다.

소개

Object Storage다량의 파일을 안전하고 효율적으로 보관, 사용이 가능한 파일 서비스 플랫폼

Object Storage는 많은 수량의 파일을 처리하기 위한 분산 기술과
불필요한 파일을 보다 경제적으로 티어링/아카이빙하는 기술입니다.
수십 년간 축척 되어온 레거시 보안 기술로 고객님의 소중한 데이터를 안전 히 보관하세요.

  • 효율적인 데이터 보호 많은 수량의 파일들을 처리하는 데 한계가 있는 기존의 백업 방식 대신 여러 개의 사본을 분산배치,
    버전을 기록하여 데이터를 보호하는 기술을 제공합니다.
  • 꼼꼼한 무결성 검증 보관되어 있는 파일들의 손상 정도를 파악하며, 과거 데이터와 비교,
    검증을 통한 높은 수준의 무결성 테스트 서비스를 제공합니다.
  • 사용하기 쉬운 사용자 UI 보관된 파일들이 필요할 경우 즉시 쉽게 사용 가능하도록 하는 검색 기능과 표준 서비스 인터페이스/ 프로토콜 구현, 서비스 네트워크 구성과
    모든 기능들이 원활하게 동작하도록 모니터링 하고 설정하고 관리해 주는 기능들이 추가되어 통합 구현이 가능합니다.
상세 기능
효율적인 데이터 보호 용
많은 수량의 파일들을 처리하는 데 한계가 있는 기존의 백업 방식 대신 여러 개의 파일 사본을 분산 배치 또는 이전 버전을 기록하여 데이터를 보호하는 기술을 제공합니다.
꼼꼼한 무결성 검증
보관되어 있는 파일들의 손상정도를 파악하며, 과거 데이터와비교 검증을 통해 높은 수준의 무결성 테스트 서비스를 제공합니다.
사용하기 쉬운 사용자 UI
보관된 파일들이 필요할 경우 즉시 쉽게 사용 가능하도록 하는 검색 기능과
표준 서비스 인터페이스/프로토콜 구현, 서비스 네트워크 구성과 모든 기능들이 원활하게 동작하도록
모니터링하고 설정하고 관리해 주는 기능들이 추가되어 통합 구현이 가능합니다.
Rados G/W를 이용한 유연한 사용
Rados G/W는 FastCGI 모듈로 구성되어 있으며 librados와 상호 작동합니다.
Openstack Swift와 Amazon S3와 호환이 가능합니다.
Ceph 클라이언트 사용 가능
Rados G/W는 Ceph 파일서비스 클라이언트를 이용하여 기존에 사용하던 Rados 클러스터에 저장을 할 수 있습니다.
Amazon S3와 Swift APIs와의 호환
S3와 Swift APIs는 공통의 namespace를 공유하며, 이를 통해서 하나의 API로 데이터를 작성할 수 있으며 다른 데이터들도 유용하게 작업 가능합니다.
