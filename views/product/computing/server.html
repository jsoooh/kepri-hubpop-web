Server

고품질의 클라우드 서버를 제공하는 서비스로 웹 인터페이스를 통해 쉽고
빠르게 다양한 서버를 구성할 수 있습니다.

소개

빠르게 생성하여 원하는 분야에 효율적으로 활용하는 서버

물리적인 서버 자원을 별도로 구매하지 않고, 클라우드 환경에서 빠르게 생성해
사용한 만큼만 비용을 지불하면 되는 효율적인 상품입니다.
인프라 운영에 대한 부담을 덜고자 하거나, 일시적으로 많은 서버 자원이 필요한 경우에 적합합니다.

  • 기업 비즈니스 형 관리방안 다양한 서비스를 개별적으로 분리하여 구성할 수 있으며,
    독립된 영역에는 기업 내에 관련한 사용자로
    구성할 수 있습니다.
  • 민첩한 컴퓨팅 서버 기업에서 필요한 컴퓨팅 서버를 빠르게 생성할 수 있으며,
    서버를 확장 축소 및 같은 유형의 서버를 한번에
    생성할 수 있습니다.
  • 고성능 고품질의 환경 최신 H/W 장비로 구성된 컴퓨팅 자원과 국내 최초 SDN/NFV 기술을 통하여 기업별로 유연한 구성이 가능한
    네트워크 환경을 지원합니다.
  • 비용 절감 및 계획적인 비용 관리 필요한 용량만큼 사용하고 비용 지불하는 서비스와
    기업별 용량계획 기준의 월 과금 방식을 적용하여 계획적인
    비용관리와 비용절감이 가능합니다.
상세 기능
초기화 스크립트 지원
‘초기화 스크립트’는 서버가 생성되는 시점에 사용자가 원하는 소프트웨어 설치 및 설정 등을 위해 필요한 정보를 서버에 전달하여 자동으로 수행하는 방법입니다.
쉘 스크립트에 익숙한 경우 스크립트를 전달하여 서버 시작 시에 실행하는 방법이 가장 쉬운 방법입니다.
서버 스냅샷 및 이미지(템플릿) 서비스 지원
‘스냅샷’은 서버나 볼륨의 현재 상태를 사진을 찍어 놓듯이 그대로 저장해 두고, 사용자가 원하면 언제든지 스냅샷을 저장한 시점 상태 그대로 돌아갈 수 있도록 해 주는 서비스입니다.
운영 중에 서버에 문제가 발생하였을 때 그 이전 상태로 복원할 수 있어 백업으로 사용되며,
서버 별로 동일하고 반복적인 작업들을 사전에 설치 및 설정한 후에 스냅샷으로 만들면 여러 대의 서버를 생성할 때 이미지(템플릿)으로도 활용할 수 있습니다
보안정책 지원
‘보안정책’은 서버에 대한 트래픽을 제어하는 방화벽 역할을 수행합니다.
서버를 생성할 때 하나 이상의 보안 그룹을 서버와 연결합니다. 보안정책에 정의된 규칙은 언제든지 수정할 수 있으며 수정된 규칙은
해당 보안정책을 사용하는 모든 서버에 자동으로 즉시 적용이 됩니다.
네트워크 서비스
공인 IP 지원 컴퓨팅 서버를 생성하게 되면 사설 IP만 할당되며 서버 내에서 인터넷 환경으로 접속은 되지만, 인터넷에서 직접 서버로 접근할 수는 없습니다.
컴퓨팅 서버를 통해 인터넷 서비스를 하기 위해서는 항시 사용자가 접근할 수 있는 고정된 공인 IP가 필요합니다.
고객은 지정한 서버에 고정된 공인 IP를 할당할 수 있으며 필요에 따라 기존의 공인 IP를 다른 서버에 할당할 수 있습니다.
(신규 서버로의 이전 경우 서비스 중단 시간을 최소화할 수 있습니다.)
기본 네트워크 지원 인터넷으로 연결이 가능한 기본 네트워크를 제공합니다.
기본 네트워크에 연결되어 있는 컴퓨팅 서버들은 모두 인터넷으로 접근 가능합니다. 고객은 사용 용도(WEB, WAS, DB 서버 등)에 따라서 기본 네트워크를 여러 개 만들어 사용할 수 있습니다.
사설 네트워크 지원 지정된 서버들 간에서만 통신 할 수 있는 독립된 네트워크 제공합니다.
클러스터링 솔루션 등과 같이 별도의 통신 채널(Health 체크, 데이터 복제 등)이 필요한 소프트웨어를 사용할 경우사설 네트워크를 사용할 수 있습니다.
서버 타입
사용 사례에 맞게 최적화된 다양한 인스턴스 유형을 제공합니다.
인스턴스 유형은 CPU, 메모리, 스토리지 및 네트워킹 용량의 다양한 조합으로 구성되며, 애플리케이션에 따라 적합한 리소스 조합을 선택할 수 있는 유연성을 제공합니다.
각 인스턴스 유형은 하나 이상의 인스턴스 크기를 포함하고 있으므로 목표로 하는 워크로드 요구 사항까지 리소스를 확장할 수 있습니다.
서버 타입 목록
타입 적용 설명
Standard(s) 범용 대부분의 용도에 적합하도록 컴퓨팅, 메모리 자원을 균형 있게 구성한 사양으로 비용이 가장 저렴한 범용적인 유형입니다.
High Computing(c) 컴퓨팅 집약적 업무용 컴퓨팅 파워를 많이 사용하는 업무에 활용할 수 있도록 최적화 된 사양으로 “Standard” 타입보다 더 많은 CPU 가 배정됩니다.
High Memroy(m) 메모리 집약적 업무용 고성능 데이터베이스, 인 메모리 데이터베이스 등 메모리를 많이 사용하는 앱에 활용 할 수 있도록 제공되는 사양입니다. “Standard” 타입보다 더 많은 메모리가 배정됩니다.
