웹 개발 도구 서비스

개발자들이 좀 더 편리하게 개발 할 수 있는 환경을
위한 개발 IDE 서비스를 제공합니다.

소개

언제 어디서든 개발이 가능한 웹 개발 도구 서비스

개발자들이 좀 더 편리하게 개발을 할 수 있는 환경을 지원해주는 IDE 서비스로서,
워크스페이스를 서버가 관리 하기 때문에 뛰어난 보안성과 용이한 개발 소스 통합이 가능하며,
인터넷만 사용 가능하면 언제 어디든 개발이 가능한 웹 개발 도구 서비스입니다.

  • 간편한 개발 환경 인터넷이 사용 가능한 환경에서,
    설치 없이 간편하게 개발할 수 있는 환경을 제공합니다.
     
  • 효율적인 소스 통합 프로젝트 공유, 멤버를 관리하며 작업 영역을
    설정하여 팀원들이 작업 할 수 있는 공간을 제공합니다.
     
  • 뛰어난 보안성 개발 워크스페이스를 서버가 관리함으로써,
    소스코드 보안에 대하여 강화된 환경을 제공합니다.
     
  • 자유로운 개발 자동 완성과, 디버깅 기능을 통하여
    기존 사용하던 IDE툴과 동일한 기능을 제공합니다.
     
상세 기능
익숙한 개발 환경 지원
웹 개발 도구 서비스는 기존 IDE 개발 도구가 지원하는 기능을 개발자들에게 제공하고 있습니다.
간편한 개발 환경 지원
설치가필요 없이 인터넷이 연결된 환경에서 바로 사용할 수 있는 개발 도구 서비스를 개발자들에게 제공하고 있습니다.
강력한 보안
개발 소스들을 관리하는 워크스페이스를 서버가 관리함으로써 개발에 관한 소스코드 유출 피해를 줄일 수 있도록 보안 환경을 제공하고 있습니다.
미리보기 환경 지원
플젝트 및 작업 영역을 실행 결과 터미널 및 웹페이지를 실행하는 미리보기 환경을 지원함으로써 프로젝트의 예상 결과물과 이슈를 미리 파악하고
프로젝트의 품질을 높일 수 있는 기능을 제공하고 있습니다.
디버깅 환경 지원
기존 IDE 툴과 같은 기능으로 개발자가작업하다 발생한 이슈에 대한 디버깅 작업을 지원하여 프로젝트를 원활하게 진행 할 수 있도록 디버깅 환경을 지원하고 있습니다.
