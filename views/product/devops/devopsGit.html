형상관리 서비스

git 기반의 소스 및 버그 관리 시스템으로써 개발자들은 등록된
이슈, 버그를 확인 후 빠르게 응답할 수 있습니다.

소개

빠르고 안전하게 사용하는 형상관리 서비스

Git 기반의 소스 및 버그 관리 시스템으로써
업그레이드가 용이하고 빠르고 쉽게 환경을 구성 할 수 있는 상품입니다.
팀 프로젝트에 대한 버전관리 및 이슈와 버그확인에 대하여 효율적인 서비스입니다.

  • 손쉬운 사용 Git 기반 시스템으로서 손쉽고 간편하게
    업그레이드와 환경구성을 마칠 수 있으며, 어느 플랫폼에서도
    지원이 가능합니다.
  • 효율적인 프로젝트 관리 형상관리 서비스를 통하여,
    프로젝트에 대한 체계적인 업무 분담과 소스 관리가
    효율적으로 반영됩니다.
  • 원활한 커뮤니케이션 소스 및 버그 관리 시스템을 사용함으로써,
    개발자들은 등록된 소스에서 발생한 이슈에 대해
    빠른 피드백이 가능합니다.
  • 민첩한 협업 지원 형상관리 서비스를 통하여,
    프로젝트를 언제 어디서든 접속하여 프로젝트를
    진행 할 수 있습니다.
상세 기능
Git에 대한 높은 호환성
Git기반의 소스 및 버그 관리 시스템으로써 모든 Git 명령어를 지원하며,기존에 사용하던 Git과 연동이 손쉽게 가능합니다.
편리한 관리
팀 별로 분리된 저장소를 제공하고, 팀 별로 제공한 쿼터에 따라 관리 가능한 저장소 수를 제한 할 수 있습니다.
원활한 커뮤니케이션
개발자들은 등록된 이슈를 쉽게 확인 할 수 있고, 버그 수정 후 빠르게 응답할 수 있도록 서비스를 제공하고 있습니다.
커밋 히스토리와 변경사항 안내
프로젝트에 대한 상세한 커밋 히스토리를 제공하며, 커밋 메시지 뿐만 아니라 소스코드 변경사항 또한 편리하게 안내 받을 수 있습니다.
유용한 접근성
프로젝트에 참여하고 있는 사용자는 형상관리 서비스를 통하여 언제 어디서든 빠르고 안전하게 자신의 소스에 접근할 수 있습니다.
마크다운 뷰 제공
형상관리 서비스는 완벽한 마크다운을 제공합니다. 현재 프로젝트 위치의 마크다운 파일을 저장하면 마크다운 문법이 적용된 화면을 제공합니다.
